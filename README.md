
## Grub2 Themes

Flat Design themes for Grub2.

## Install

Usage:  `./Install`  **[OPTIONS...]**

|  OPTIONS:      | |
|:---------------|:-------------|
| -v, --vimix    | Vimix grub theme |
| -s, --stylish  | stylish grub theme |
| -t, --tela     | tela grub theme |
| -s, --slaze    | tela grub theme |
| -h, --help     | Show this help |
